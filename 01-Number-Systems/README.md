# Module 01 – Number Systems

This module covers the fundamentals of number systems and arithmetic operations used in Digital Logic Circuits.

## Syllabus Coverage

1. **Introduction**
   - Need for number systems
   - Role of number systems in digital circuits

2. **Positional Representation of Numbers**
   - Decimal number system
   - Binary number system
   - Octal number system
   - Hexadecimal number system

3. **General Radix (r) Number System**
   - Positional weight
   - Representation of numbers in base-r

4. **Conversions Between Number Systems**
   - Decimal ↔ Binary
   - Binary ↔ Octal
   - Binary ↔ Hexadecimal
   - General radix conversion methods

5. **Complements**
   - (r−1)’s complement
   - r’s complement
   - Signed and unsigned number representation

6. **Arithmetic Operations**
   - Binary addition
   - Binary subtraction
   - Use of complements for subtraction
   - Overflow detection and handling

7. **Binary Codes**
   - BCD (Binary Coded Decimal)
   - Excess-3 code
   - Gray code
   - ASCII code
   
## Learning Outcomes
- Convert numbers between different bases
- Perform binary arithmetic operations
- Understand signed number representation techniques
